# 💰 Finance Analytics Dashboard

An interactive **Finance Analytics Dashboard** built in **Microsoft Excel** to monitor key financial metrics, analyze business performance, and support data-driven decision-making. The dashboard provides real-time insights into revenue, expenses, profit, transaction trends, and financial performance through dynamic visualizations.

---

## 📌 Project Overview

Organizations often struggle with scattered financial data and time-consuming manual reporting. This project transforms raw financial transaction data into an interactive dashboard that helps stakeholders quickly monitor financial performance and identify business trends.

---

## 🎯 Business Problem

- Financial data was distributed across multiple records.
- Manual reporting was time-consuming and prone to errors.
- Decision-makers lacked a centralized view of key financial KPIs.

---

## ✅ Solution

Developed an interactive Excel dashboard that:

- Tracks Revenue, Expenses, and Profit & Loss (P&L)
- Monitors financial performance through KPI cards
- Enables filtering using interactive slicers
- Provides dynamic charts for better business insights
- Automatically updates reports using Pivot Tables and Excel formulas

---

## 📊 Dashboard Features

- 💵 Total Revenue
- 💸 Total Expenses
- 📈 Profit & Loss Analysis
- 📊 Revenue by Department
- 💳 Payment Method Analysis
- 📂 Category-wise Financial Performance
- 🔄 Transaction Type Analysis
- ✔️ Approval Status Distribution
- 🎛 Interactive Slicers
- 📉 Dynamic Charts & KPI Cards

## 🚀 How to Use

1. Download the Excel workbook.
2. Open it in Microsoft Excel (2019/2021 or Microsoft 365 recommended).
3. Enable editing if prompted.
4. Use the slicers to filter data.
5. Explore KPIs and charts for financial analysis.
